\subsection{ADMN-IU02.2 Registrar nuevo Colaborador}

\subsubsection{Objetivo}
En la figura \ref{ADMN-IU02.2} se muestra la interfaz correspondiente con la funcionalidad descrita en las
trayectorias del caso de uso \refElem{ADMN-CU02.2}, la cual permite al actor crear una cuenta nueva en el sistema para un nuevo usuario que actué como colaborador
dentro del mismo.
A traves de esta pantalla se generan las nuevas contraseñas o credenciales de acceso para el sistema.

\subsubsection{Acciones}
\Titem \IUbutton{Aceptar} : Al presionar el botón, genera las nuevas credenciales para acceder al sistema y las envía por correo.
\Titem \IUbutton{Cancelar} : Al presionar el botón, no realiza ninguna acción sobre el registro y por ende no crea ninguna cuenta.
